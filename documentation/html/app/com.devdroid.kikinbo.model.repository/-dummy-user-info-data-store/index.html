class DummyUserInfoDataStore

A dummy data store for storing mock user information.

This class provides a set of predefined UserDataModel objects representing dummy users with basic information such as user ID, username, email, and phone number. It is used for testing purposes and simulates a data source that could be replaced with a real backend or database in a production environment.

Each user is represented by a UserDataModel with the following properties:

  • userId: Unique identifier for the user.

  • username: Full name of the user.

  • email: Email address of the user.

  • phoneNumber: Contact number of the user.
